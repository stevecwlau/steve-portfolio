Project information

  • Title: Spatio-temporal and static heatmaps of active fire during the 2019-2020 bushfire crisis in Australia
  • Data source: Fire Information for Resource Management System (FIRMS), processed by LANCE using the standard MODIS MOD14/MYD14 Fire and Thermal Anomalies product. Data is collected from Aug, 2019 to Dec, 2019.
  • Software used: ArcGIS Pro
  • Skills: Spatio-temporal data modelling, heatmap, visualization, animation

The active fire dataset was collected from the Fire Information for Resource Management System (FIRMS). Each active fire location represents the centroid of a 1km pixel that is flagged by the algorithm as containing one or more fires within the pixel. The original dataset contains all the active fire spots (hourly) for the year of 2019, so it is a pretty large dataset to begin with (over 400k points).

In this project, I first built a geoprocessing model to extract points from collected from August, 2019 to December, 2019 (see Picture 3), as the goal of this project is to study pattern during this period of bushfire crisis. As shown in Picture 2, the point data was converted to a heatmap to visualize clusters of fire events during this 4 months. The heatmap was weighted by the brightness temperature in Kelvin which tells us whereabout the fire is most intense and dangerous. Other than the static heatmap, I was able to build a spatio-temporal map to study the movement of fire events across the whole continent by using the ArcGIS Pro animator tool (see Picture 1).
